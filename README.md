# fruit_cehck
基于卷积神经网络搭建的《水果识别系统》

## 介绍
水果识别系统，使用Python作为主要开发语言，基于深度学习TensorFlow框架，搭建卷积神经网络算法。并通过对5种垃圾数据集进行训练，最后得到一个识别精度较高的模型。并基于Django框架，开发网页端操作平台，实现用户上传一张图片识别其名称。
